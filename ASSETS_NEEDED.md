## üìù Notes

1. **Image Specifications:**
   - All images should be optimized for web (compressed but high quality)
   - Use WebP format when possible for better performance
   - Maintain aspect ratios for consistency

2. **Brand Consistency:**
   - All images should match the brand colors (#0000ff blue)
   - Maintain professional, modern aesthetic
   - Ensure images reflect NZ/Auckland context where relevant

3. **Legal Considerations:**
   - Ensure all images have proper licensing
   - Get permission for any customer photos used
   - Use stock photos with appropriate licenses

4. **Accessibility:**
   - All images should have descriptive alt text
   - Ensure images don't contain important text (use HTML text instead)
